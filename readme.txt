Simple Textile provides the option to format posts or pages using the Textile markup language.

== Description ==

Simple Textile is a filter for posts and pages that can be used to write posts in Textile. Simply tick the "Process in Textile"
checkbox while writing your latest blog entry, and it will be automatically parsed through Textile. Disable the visual editor from your
user settings.

Problems, questions or feedback? Send us mail at [contact@rojekti.fi](mailto:contact@rojekti.fi)

== Installation ==

1. Upload the simpletextile directory to your plugins directory or install it through the dashboard
2. Activate it
3. Ensure that posts or pages that need to be Textilized have the Textile checkbox checked
